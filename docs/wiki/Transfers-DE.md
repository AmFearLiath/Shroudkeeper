# Transfers (DE)

Transfers kopieren Save-Rolls zwischen Singleplayer und Multiplayer.

## 1) Transfer vorbereiten

1. Quelle und Ziel festlegen.
2. Slot und Roll auswählen.
3. Prüfen, dass Quelle und Ziel nicht identisch sind.
4. Bei Ziel Multiplayer: Server stoppen.
5. Backup erstellen.

## 2) Transfer starten

1. **Transfer starten** klicken.
2. Hinweise/Bestätigungen prüfen.
3. Statusmeldungen verfolgen (Vorbereitung, Kopieren, Index schreiben, fertig).

## 3) Nachkontrolle

- Zielseite neu scannen.
- Prüfen, ob `latest` und Roll-Dateien korrekt übernommen wurden.
- Bei Problemen: auf letztes Backup zurückgehen.

## Typische Fehler

- Kein aktives Profil (bei Multiplayer).
- Spiel/Server läuft noch.
- Fehlende Rechte auf Zielpfad.
- Ungültige Richtung oder unvollständige Auswahl.
