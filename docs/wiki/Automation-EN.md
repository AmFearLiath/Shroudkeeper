# Automations (EN)

Automations run recurring jobs on a schedule.

## 1) Create a job

1. Open **Automations**.
2. Click **Add job**.
3. Configure name, job type, and schedule.
4. Select profile/source/target as needed.
5. Save.

## 2) Schedules

- Daily
- Selected weekdays
- Custom time setup

## 3) Operation

- Start/stop scheduler
- Use **Run now** for manual validation
- Check last run and status fields

## 4) Common pitfalls

- Missing credentials
- Invalid source folder
- Job already running
- Game still running during local-source upload

## Recommendation

- Manually validate each new job first.
- For critical workflows, establish regular review checks.
