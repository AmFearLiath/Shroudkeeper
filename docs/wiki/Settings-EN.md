# Settings (EN)

This page covers remaining global settings.

## 1) Language

- Select language in **Settings**.
- Verify key UI elements after switching.

## 2) Theme

- Choose a theme in **Settings**.
- Theme applies app-wide.

## 3) Backup directory

- Set backup target folder.
- Ensure enough free space and write permissions.

## 4) Singleplayer Save Root

- If auto-detect is incorrect, set path manually.
- Run scan after path changes.

## 5) Live log/status

- Live log shows running operations and errors.
- For issues, inspect the latest relevant log entry first.

## 6) Data storage

- App data: `%APPDATA%/Shroudkeeper`
- No persistent user data intended in installation directory.
